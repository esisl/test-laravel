# Тестовое задание: Сбор данных и доступ к БД

## Архитектура
- **Стек**: Laravel 8, PHP 8.1, Docker Compose, MySQL 8.4
- **Контейнеры**: `app` (PHP-FPM), `db` (MySQL), `nginx`
- **Импорт данных**: `php artisan api:fetch --entity=<sales|orders|incomes>`

## Доступ к базе данных
